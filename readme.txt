== Description ==

Responsive Attention Box  is a plugin that helps push important messages onto the screen
and your users.Images and jQuery have been used to enhance the User Experience.
You can easily display custom texts, Twitter posts RSS feeds etc .


== Installation ==

1. Upload the 'attention-boxes' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Page or Posts and use  shortcodes


[label style="success"]Well done! You successfully read this important alert message. <span class="close">&nbsp;</span>[/label]
[label style="info"]Head up! This alert needs your attention, but its not super important. <span class="close">&nbsp;</span>[/label]
